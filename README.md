本文件为SZTU三维感知技术实验室暑期进行的结构光三维重建项目中Basler相机与DLP4500投影仪与GCD平移台实现同步的代码文件。同步精度可达微秒级。
具体实现都在DLP4500FINAL文件夹里的main.cpp文件里，其它的杂糅文件都是Basler工业相机运行所必需的文件，来自官网下载的相机驱动SDK文件。
要结合使用DLP4500生产厂商提供的LightCrafter软件来设置DLP4500投影仪的图片烧录，播放序列以及投影曝光长度等一系列参数。
GCD平移台为大恒光电公司旗下的一款平移台系列产品，通过串口通信来控制即可。
